# Python Labs

Repositório com experimentos, provas de conceito e pequenos projetos desenvolvidos em Python durante estudos e aplicações práticas.

O objetivo deste repositório é reunir soluções para desafios relacionados à engenharia de dados, análise de dados, automação de processos e exploração de novas bibliotecas.

## Estrutura do repositório

```
python-labs/
│
├── dashboards/         # Dashboards interativos
├── pdf-processing/     # Extração e transformação de documentos PDF
├── data-analysis/      # Projetos de análise exploratória de dados
├── case-studies/       # Estudos de caso e desafios técnicos
└── utilities/          # Scripts utilitários e funções reutilizáveis
```

## Temas abordados

- Análise de dados com Python
- Automação de processos
- Processamento de documentos
- Dashboards interativos
- Consumo de APIs
- Manipulação de arquivos
- Exploração de bibliotecas
- Jupyter Notebooks

## Tecnologias

- Python
- Pandas
- NumPy
- Plotly Dash
- pdfplumber
- Jupyter Notebook

## Objetivo

Este repositório funciona como um laboratório de desenvolvimento, reunindo estudos, experimentos e pequenas soluções criadas para explorar diferentes abordagens na resolução de problemas utilizando Python.

Os projetos aqui publicados não representam aplicações completas, mas sim experimentos que contribuíram para o desenvolvimento de habilidades técnicas e para o aprofundamento em temas relacionados à Engenharia de Dados, Analytics e Automação.

# Python Labs

Collection of small projects, experiments and proof-of-concepts developed while studying and applying Python to real-world data challenges.

This repository serves as a technical laboratory for testing ideas, exploring libraries and building reusable solutions related to data engineering, automation and data analysis.

## Repository Structure

```
python-labs/
│
├── dashboards/          # Interactive dashboards and data visualization
├── pdf-processing/      # PDF extraction and document processing
├── data-analysis/       # Exploratory data analysis projects
├── case-studies/        # Technical challenges and small case studies
└── utilities/           # Reusable scripts and helper functions
```

## Topics Covered

- Data analysis with Python
- Data processing and automation
- PDF extraction and transformation
- Interactive dashboards with Dash
- Data visualization
- API consumption
- Jupyter Notebook experiments

## Technologies

- Python
- Pandas
- NumPy
- Plotly Dash
- pdfplumber
- Jupyter Notebook

## Purpose

Rather than production-ready applications, this repository contains learning projects, prototypes and experiments that helped develop practical skills in data engineering and analytics.

As new topics are explored, this repository will continue to grow with additional experiments and reusable solutions.
